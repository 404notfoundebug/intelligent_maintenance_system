# 前端项目

面向电梯扶梯维保场景的多模态知识检索与标准化作业辅助系统前端，基于 Vue3 + Vite + Element Plus。

## 安装依赖

```bash
npm install
```

## 启动开发环境

```bash
npm run dev
```

访问地址：

```text
http://127.0.0.1:5173
```

## 后端地址配置

开发环境后端地址配置在 `.env.development`：

```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## 默认登录账号

```text
admin / admin123456
```

## 打包

```bash
npm run build
```
